Параллельная многопроцессорная обработка данных HPC

Системы IntellectDigital SciPhi, использующие многоядерные сопроцессоры (MIC — Many Integrated Core) Intel Phi, предназначены для максимальной эффективности расчетов задач, допускающих параллельную обработку (интеллектуальная обработка фото и видео (видео-наблюдение, поиск фото и видео, сбор статистики в видео-контента), обработка медицинских изображений, математическое моделирование физических процессов). Сопроцессоры Intel Phi — это важнейшая инновация 2012-2013 гг. в индустрии HPC, применимая как для построения систем класса Top500, так и для персонального суперкомпьютера. Ключевой возможностью систем IntellectDigital SciPhi является возможность радикального ускорения счета инженерных или научных задач при минимальной оптимизации существующего кода программ, написанных на для многопроцессорных систем на языках C, C++, Fortran, с помощью дополнений к существующим компиляторам. В данном случае не требуется решать отдельную задачу под оптимизацию кода под параллельные расчеты на GPU-ядрах. Многие существующие программные решения могут быть радикально ускорены без дополнительных инвестиций в переписывание кода под GPU (что, кстати, эффективно далеко не во всех задачах).

Мы предлагаем варианты систем ID SciPhi для проектов различного размера: от отладочных задач, до больших кластеров. Комбинация мощной — 4-х процессорной системы традиционной архитектуры с многоядерным сопроцессором позволит разработчикам оптимально спроектировать приложения с точки зрения эффективности затрат и вычислительной мощности под конкретные задачи.

Уникальность многоядерной архитектуры Intel MIC в следующем:

  • Имеется много небольших, с низким энергопотреблением процессорных ядер архитектуры Intel;
  • Векторный обработчик значительно шире, чем в имеющихся многоядерных дизайнах, что дает гораздо более высокую эффективность на ватт в операциях с плавающей точкой.

Intel MIC значительно упрощает организацию параллельной обработки в имеющихся задачах, которые могут задействовать сотни ядер, и позволяет использовать 512-битную адресацию памяти для таких задач как методы моделирования Монте Карло, моделирования финансовых задач, сейсмоаналитику для задач обнаружения нефти и газа, прогнозы погоды, молекулярная динамика, создание медийного контента.

Наряду с собственными решениями, являясь партнером IBM Premier, мы можем предложить для HPC задач комплексное решение IBM® Platform™ HPC для IBM System x® и Intel® MIC.

IBM Platform HPC содержит полный набор возможностей управления:

  • Управление кластером (Cluster management)
  • Управление нагрузками (Workload management)
  • Мониторинг и подготовка отчетов загрузки (Workload monitoring and reporting)
  • Системный мониторинг (System monitoring and reporting)
  • Динамический выбор ОС для загрузки (Dynamic operating system multi-boot)
  • Библиотеки MPI (IBM Platform MPI)
  • Подготовленные шаблоны для приложений (Integrated application scripts and templates)
  • Универсальный WEB-доступ (A unified web portal)

На базе платформ IBM System x iDataPlex® dx360 M4 и Intel Xeon Phi под управлением IBM Platform HPC мы готовы построить многопользовательскую систему под решение самых сложных задач HPC.
