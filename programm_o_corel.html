Программное обеспечение Corel

CorelDRAW Graphics Suite 2017

Универсальное и мощное программное обеспечение для графического дизайна
Представляем CorelDRAW Graphics Suite 2017 — наше новейшее и самое передовое программное обеспечение для графического дизайна. В состав пакета входят все признанные в индустрии инструменты, представленные в предыдущей версии CorelDRAW Graphics Suite, а также множество других функций.

Corel AfterShot Pro 3

Наилучшие снимки. Еще быстрее
Corel® AfterShot™ Pro 3 — новая версия программы для профессиональных фотографов и фотолюбителей для редактирования и каталогизации фотографий. Программа обеспечивает обработку фотографий в 4 раза быстрее, чем подобные продукты конкурентов.

CorelCAD 2017

САПР-программа для 2D проектирования, 3D моделирования и 3D печати
Интуитивный интерфейс и полный набор настраиваемых инструментов САПР, работающих в интерактивной среде, делают CorelCAD™ 2017 самым универсальным и рентабельным решением для автоматизированного проектирования. Профессиональные инструменты обеспечивают превосходный результат при выполнении точных работ по проектированию в двух- и трехмерном режимах. CorelCAD 2017 использует .DWG в качестве собственного формата файлов, чем обеспечивает совместимость со всеми основными программами САПР и предоставляет пользователям возможность работать в привычной среде.

CorelDRAW Home&Student Suite X8

Воспользуйтесь преимуществами CorelDRAW® X8 при создании проектов для дома и учебы
CorelDRAW Home & Student Suite X8 для дома и учебы содержит простые в использовании инструменты трассировки и макетирования, а также средства для редактирования фотоснимков, представленные в Corel® PHOTO-PAINT® и Corel® AfterShot™. Разработанный для энтузиастов самых разных уровней подготовки, экономически эффективный пакет позволяет быстро и с уверенностью создавать логотипы, информационные бюллетени, доклады, фотопроекты и материалы для социальных сетей.
